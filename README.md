Gêmeo Digital: Monitoramento de Qualidade do Ar na MineraçãoEste projeto implementa uma arquitetura IoT ponta a ponta utilizando FIWARE, Node-RED e Docker para monitorizar zonas de mineração, garantindo segurança operacional através do controle inteligente de ventilação.🚀 Arquitetura do SistemaO sistema é composto por:Simulador (Python): Gera telemetria (CH4, CO, O2, etc.) para 12 sensores distribuídos em 3 zonas.Broker (Mosquitto): Barramento de mensagens MQTT.Gateway (Node-RED): Processa regras de negócio (cálculo de média de CH4) e integra com o FIWARE.Context Broker (Orion): Gêmeo Digital (NGSIv2).Persistência (QuantumLeap + CrateDB): Armazenamento de séries temporais.Interface (Streamlit): Dashboard analítico para visualização de dados e eficiência energética.📋 Pré-requisitosDocker e Docker Compose instalados.Python 3.9+ com bibliotecas instaladas (pip install streamlit pandas requests paho-mqtt).🛠️ Como Executar1. Iniciar a InfraestruturaNa raiz do projeto, suba os serviços FIWARE/MQTT:docker compose up -d
2. Iniciar o Simulador de SensoresAbra um terminal e execute o publicador de dados:python publisher.py
3. Configurar o Fluxo no Node-REDAcesse http://localhost:1880.Importe o conteúdo do arquivo FluxoNodeRED.json (Menu > Import).Verifique se o nó MQTT está conectado (bolinha verde).Clique no botão vermelho Deploy no canto superior direito.4. Executar o DashboardEm um novo terminal, inicie a interface de monitoramento:streamlit run dashboard.py
Acesse http://localhost:8501 no seu navegador.🔧 Diagnóstico de ProblemasSe os dados não aparecerem, utilize o botão "🛠️ Executar Diagnóstico do Sistema" dentro do próprio Dashboard Streamlit. Ele valida automaticamente:Conexão com o Orion.Status das Subscriptions.Integridade da gravação no CrateDB.📦 Estrutura de Arquivosdocker-compose.yml: Definição de todos os contêineres.publisher.py: Simulador MQTT.dashboard.py: Interface de análise.FluxoNodeRED.json: Lógica de integração e regras de ventilação.RegraDeNegocio.js: Script de cálculo de médias dentro do Node-RED.Projeto desenvolvido para a disciplina de Desenvolvimento IoT - IFRN Campus Parnamirim.
